CollectionViewDriverOptions

public struct CollectionViewDriverOptions : Hashable
extension CollectionViewDriverOptions: CustomDebugStringConvertible

Defines various options to customize behavior of a CollectionViewDriver.

  • reloadDataOnReplacingViewModel

    Specifies whether or not the CollectionViewDriver should perform a hard reloadData() when replacing the CollectionViewModel with a new one, or if it should always perform a diff.

    A replacement occurs when providing a new CollectionViewModel to the CollectionViewDriver that has a different id than the previous model.
